
# GPA & CGPA Calculator

A console-based GPA and CGPA Calculator built with Python.

## Features

* Calculates grades automatically from scores
* Calculates quality points
* Calculates semester GPA
* Calculates cumulative GPA (CGPA)
* Supports multiple semesters
* Validates score input (0–100)

## Technologies Used

* Python

## How to Run

1. Install Python
2. Download the project
3. Run:

 CGPA_calculator.py
